# 🍽️ Responsive Restaurant Website

A lightweight, clean, and fully responsive landing page for a restaurant, built using pure HTML and CSS.

## 🚀 Project Overview
This project is a front-end demonstration of a restaurant website. The goal was to create a visually appealing user interface that works seamlessly across all device sizes (mobile, tablet, and desktop).

## ✨ Features
- **Fully Responsive Design:** Optimized for mobile, tablet, and desktop screens using CSS Media Queries.
- **Modern UI/UX:** Clean layout with a focus on food imagery and easy navigation.
- **Navigation Bar:** A smooth navigation menu for different sections (Home, Menu, About, Contact).
- **Menu Section:** A well-structured layout to display food items and prices.
- **Contact Section:** A simple area for location and contact details.

## 🛠️ Technologies Used
- **HTML5:** For semantic web structure.
- **CSS3:** For styling, Flexbox, and Grid layouts.
